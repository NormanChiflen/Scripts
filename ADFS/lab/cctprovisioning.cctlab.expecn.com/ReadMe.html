This ZIP files contains several files that may or may not need to be used for your particular installation.  Please see https://certs/faq for installation tips.

<host>.crt

This is the Base64 encoded certificate file provided by the certificate authority.  It only contains the public key and is safe to be distributed to vendors if this request is being used for client/server authentication.

<host>.pem

This is the Base64 encoded certificate file that contains the private key and all the certificates from the certificate authority.  It is typically used for conversion purposes for different applications. 

<host>.pfx

Windows can use this file to install both public and private keys as well as the certificate chain.  It is a binary file that is DER encoded.

EntrustRootCA.crt

This file is our vendor supplied public certificate.  It should be installed in the Enterprise certificate store.

EntrustChainCa.crt

This file is our vendor supplied public chain certificate.  It should be installed in the Enterprise chain store.  We need this because is is the actual authority that issues our certificates.

InternalChain.crt and InternalRoot.crt

If your certificate was internally generated, this was the infrastructure that did it.  Both certificates need to be loaded in the Enterprise store.  These are both base 64 encoded and PEM formatted.  Please see the FAQ page noted above for instructions.

MSChain.p7b

This is another format of our internal PKI's authorities certificate list. 

netscaler.txt

This is a file necessary for installation on a Citrix NetScaler.  All others installations can ignore it.
